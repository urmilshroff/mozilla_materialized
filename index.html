What is Open Source?
At this point, you might be wondering: what is Open Source software all about?

Well, Open Source isn't just about free software. It's about projects where the code is made publicly available (usually on a site called GitHub) so that a very healthy community of developers from around the world can collaborate and contribute to it by suggesting improvements and bugfixes. It's like writing a book where many other authors can pitch in some great ideas for the story.

So why is it so important? Because it helps push the software industry forward more than anything else. The most impactful and influential software developments in the world today such as Android, Linux, TensorFlow, React and Python are all completely Open Source.

These projects haven't been developed only by a single person or organization. They all have contributions from the public community, where people (just like you!) have helped build, test, secure and improve the quality of the software to make it as big and as awesome as it is today.
